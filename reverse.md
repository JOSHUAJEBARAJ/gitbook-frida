## Reversing the app

1) Clone the repo
```
git clone https://github.com/b-mueller/apkx
```
2) Move into the cloned repo
```
cd apkx
```
3)Install the app
```
sudo ./install.sh

```

## Verification

Run the following command to verify the installation
```
apkx -h
```
![Verification](ap-ver.jpg)

## Usage

```
apkx [appname]

```
